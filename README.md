# Blotz Training Onboarding

This repository contains all the training modules for Frontend, Mobile, and Backend development.

## Project Structure

To keep things organized, the projects are grouped by track:

### 🌐 Frontend Track

Located in `/frontend`

- **[Basics](./frontend/basics)**: HTML, CSS, and JavaScript fundamentals.
- **[Todo App](./frontend/todo-app)**: The main Next.js project.
  - _Includes: React, Next.js, ShadCN UI, React Hook Form, and Tanstack Query._

### 📱 Mobile Track

Located in `/mobile`

- **[Basics](./mobile/basics)**: Initial Expo and React Native learning.
- **[Todo App](./mobile/todo-app)**: The feature-based mobile app.
  - _Includes: Expo Router, Feature-based architecture, and React Query._

### 🖥️ Backend Track

Located in `/backend`

- Upcoming modules for C#, .NET, and API development.
